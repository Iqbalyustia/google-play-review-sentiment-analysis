# Google Play Review Sentiment Analysis

A Streamlit-based application for scraping Google Play reviews, preprocessing Indonesian text, performing sentiment analysis using VADER/TextBlob, and classifying reviews into PIECES domains.

## Features

- Google Play Review Scraping
- Indonesian Text Preprocessing
- Sentiment Analysis
- PIECES Domain Classification
- Streamlit Dashboard

## Tech Stack

- Python
- Streamlit
- Pandas
- Google Play Scraper
- Sastrawi
- VaderSentiment
- TextBlob
- Matplotlib
- Seaborn

## Installation

```bash
pip install -r requirements.txt
streamlit run app.py
